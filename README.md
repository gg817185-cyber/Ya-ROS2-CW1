Интсрукция к пакету exam_robot

Сборка пакета
Перейдите в корень вашего воркспейса и выполните сборку:
cd ~/ros2_ws
colcon build --packages-select exam_robot --symlink-install

Настройка окружения
После сборки обязательно обновите переменные окружения:
source install/setup.bash

Запуск всей системы
Для одновременного запуска всех узлов (контроллера, датчиков, дисплея и модели робота) выполните команду:
ros2 launch exam_robot robot_system.launch.py

Проверка работы через топики
Чтобы убедиться, что данные передаются корректно, откройте новый терминал, сделайте source и используйте следующие команды:

Просмотр общего статуса:
ros2 topic echo /robot_status

Просмотр уровня заряда батареи:
ros2 topic echo /battery_level

Просмотр расстояния до препятствия:
ros2 topic echo /distance

Просмотр команд скорости:
ros2 topic echo /cmd_vel

Система работает автоматически: при снижении заряда батареи или приближении к препятствию робот будет менять режим движения (замедляться, поворачивать или останавливаться), что будет отображаться в логах терминала.
